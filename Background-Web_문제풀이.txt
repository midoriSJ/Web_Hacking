Background-Web 부분을 다 읽어보고 실습 문제인 devtools-sources라는 문제가 있어 풀어보았다

개발자 도구의 Sources 탭 기능을 활용해 플래그를 찾아보세요.

플래그 형식은 DH{...} 입니다.

그러면 ctrl + shift + F 해서 DH{} 양식을 찾으면 되는 것 아닌가? 라고 생각했지만
이건 나의 착각이었다.

그래서 문제 파일을 받아보았다.

바로 F12 눌러서 source 탭 입장.

ctrl + shift + F를 눌러 DH를 검색해보니

webpack/styles/main.scss 파일의 4146번째 줄에 flag를 숨겨놓았다.
